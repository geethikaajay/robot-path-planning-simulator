# robot-path-planning-simulator
Python simulation of the A* path planning algorithm for robot navigation on a grid with obstacles.
This project demonstrates the A* path planning algorithm commonly used in robotics and autonomous navigation.

The simulator represents the environment as a grid where:
- free cells represent navigable space
- blocked cells represent obstacles

The A* algorithm searches for the shortest path from a start position to a goal position using a heuristic-based search strategy.

The output visualization shows how the robot finds an optimal path while avoiding obstacles.
